Dawud Muhammed - Portfolio
Laravel Alpine.js Tailwind CSS License

A modern, responsive portfolio platform for showcasing projects, skills, and testimonials. Built with Laravel 13 for scalable backend management and dynamic content.

🚀 About High-end SaaS-style portfolio with admin dashboard, role-based access, and automated OG image generation for SEO and performance.

🛠️ Tech Stack

Backend: Laravel 13, PHP 8.3
Frontend: Alpine.js, Tailwind CSS
Database: MySQL
Other: Vite, Node.js (for OG images), Composer
📦 Features ✨ Dynamic content management (CRUD for posts/projects) 🎨 Dark/light mode with responsive design 📱 API integrations (analytics, social media) 🚀 Auto-dispatching OG images and queued mail 📊 Admin dashboard with settings and analytics

🏗️ Built With

Laravel - Backend framework
Alpine.js - Reactive frontend
Tailwind CSS - Styling
Vite - Build tool
MySQL - Database
🚀 Getting Started Prerequisites

PHP 8.3+, Composer, Node.js 18+, MySQL
Installation

git clone https://github.com/Dawud-Muhammed/portfolio.git
cd portfolio
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run build
php artisan serve
📂 Project Structure

portfolio/
├── app/             # Laravel app logic
├── resources/       # Views and assets
├── routes/          # Web/API routes
├── database/        # Migrations
├── public/          # Static files
└── vite.config.js   # Build config
🌐 Deployment Deploy on Railway, Heroku, or Nginx. Enable queues for mail and configure SSL.
